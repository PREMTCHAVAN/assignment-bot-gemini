# AI Assignment Assistant (Gemini)

An AI-powered application that helps students generate structured and well-formatted assignment answers using Google's Gemini API.

---

## Overview

This project is a Streamlit-based web application designed for Computer Engineering students. It allows users to input assignment questions and receive structured, easy-to-understand responses generated by an AI model.

---

## Features

* Structured answers (Definition, Explanation, Example, Conclusion)
* Multiple response modes:

  * Assignment (formatted answers)
  * Simple (concise explanation)
  * Detailed (in-depth explanation)
* Interactive chat interface
* Session-based chat history
* Integration with Gemini API

---

## Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/PREMTCHAVAN/assignment-bot-gemini.git
cd assignment-bot-gemini
```

---

### 2. Create a virtual environment

```bash
python -m venv env
```

Activate the environment:

**Windows**

```bash
env\Scripts\activate
```

**macOS/Linux**

```bash
source env/bin/activate
```

---

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

---

### 4. Configure environment variables

Create a `.env` file in the root directory and add your API key:

```
GEMINI_API_KEY=your_api_key_here
```

You can obtain an API key from:
https://makersuite.google.com/app/apikey

---

### 5. Run the application

```bash
streamlit run scripts/app.py
```

---

## Usage

* Enter your assignment question in the input field
* Select the desired response mode from the sidebar
* The application generates a structured response using the Gemini model

---

## Notes

* The free Gemini API tier has rate limits. If exceeded, wait a few seconds before retrying
* Do not commit your `.env` file or API key to version control
